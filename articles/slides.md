# Markdown Slides - 轻量级演示文稿解决方案

Markdown Slides 是一款基于 Web 的轻量级演示文稿生成工具。与传统 PPT 复杂的排版流程不同，它允许用户直接使用标准的 Markdown 语法编写幻灯片内容，极大地简化了演示文稿的创建过程。

**体验版地址**：<https://rectangle518.github.io/blog-passages/slides.html>

## 产品简介

Markdown Slides 的开发基于以下核心理念：

-   **极速创作**：摒弃了传统演示软件中繁琐的拖拽与精细化排版，让创作者能够像编写代码注释或撰写技术文档一样，通过纯文本的方式快速生成结构清晰的幻灯片。
-   **内容优先**：深度聚焦于技术分享、学术交流、课程讲义等场景。在这些场景中，信息的清晰传递远比复杂的动画特效更为重要。工具弱化视觉干扰，强化内容本身的逻辑性与可读性。
-   **通用兼容**：依托于成熟且开放的 Markdown 生态，它不仅支持基础的文本格式化，更完美地融合了技术写作中常用的扩展语法，如多级标题、数学公式（LaTeX）、表格以及图片、视频等多媒体资源的嵌入。

本产品旨在精准解决技术分享、学术汇报及日常交流中“重内容、轻特效”的特定需求。通过将文档写作与演示制作这两个环节合二为一，Markdown Slides 让创作者能够将全部精力专注于内容的构思与打磨，真正实现“所写即所得”的高效、纯粹的演示体验。

## 功能概述

-   **实时预览**：编辑区与预览区无缝联动，Markdown 源码的任何变更都将毫秒级同步至预览区，提供即时的视觉反馈，无需手动刷新页面。
-   **文件传输**：支持文稿的本地导入与导出功能，能够无缝衔接到用户现有的文档工作流中，极大地方便了离线编辑、内容备份与版本管理。
-   **放映展示**：提供沉浸式的全屏演示模式，并支持键盘方向键进行幻灯片导航，为观众带来专业、流畅的观看体验。
-   **云端存储**：集成云端存储能力，实现文稿在不同设备间的数据同步与即时分享，打破设备限制，让创作和演示随时随地。
-   **文档分页**：通过识别 Markdown 内容中的特定分隔符语法，系统能够自动将一篇长文智能切割为逻辑清晰的独立幻灯片页面，使内容组织更加结构化。
-   **语法支持**：完美兼容 Markdown 语法规范中的绝大部分功能，涵盖从基础的多级标题、列表、引用，到进阶的表格、图片以及复杂的 LaTeX 数学公式。
-   **主题系统**：内置多套精心设计的视觉主题（如深色模式、极简风格等），并支持一键切换。用户亦可基于 CSS 变量自定义主题，以满足不同场合下的视觉偏好。

**温馨提示**：当前提供的体验版为概念验证版本，以上部分高级功能（如云端存储、主题自定义等）并未完全实现。该产品也许会继续迭代，敬请期待后续更新。
